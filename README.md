# OpenGL Renderer
Study Project on C++/OpenGL including features like:
- Shadow mapping (directional + point lights) not yet, but gonna be soon
- Blinn-Phong / Phong shading
- Skybox (cubemap)
- MSAA (multisample anti-aliasing)  
- Instanced rendering (100k asteroids @ 180fps)
- Post-processing framebuffer
- ImGui debug panel
